# EduCounsel AI

EduCounsel AI is an AI-powered education counseling and learning support chatbot system. It helps students ask questions about courses, career paths, learning topics, and education guidance through a text-based chatbot interface.

The system provides course recommendations, career counseling, learning explanations, chat history, and a database-connected knowledge base. Students can also teach the chatbot new question-and-answer pairs, which are saved in the SQL Server database and reused in future conversations.

## Features

* AI chatbot for education counseling
* Course recommendation support
* Learning topic explanations
* SQL Server knowledge base
* User-taught learning feature
* Real-time chat history
* Course database integration
* Visual AI avatar interface
* FastAPI backend
* HTML, Tailwind CSS, and JavaScript frontend

## Technologies Used

* Python
* FastAPI
* SQL Server
* HTML
* Tailwind CSS
* JavaScript
* OpenRouter AI API
* REST API

## Project Purpose

The main purpose of EduCounsel AI is to support students who are confused about what to study, which course to choose, or how to plan their future career. The system provides simple, fast, and helpful guidance using AI and database-driven responses.
